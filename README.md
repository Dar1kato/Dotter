# Dotter

Dotter is a free to use, opensource web application that converts dot art into coordinates. Dotter serves like an easy to use method of obtaining relative placements of any substance into a disk-like container, mainly a petri dish. Dotter’s main purpose is to aid in the development of biohacking projects.

## Usage

To use Dotter:

1. Open the page on your web browser.
2. Draw you shape by clicking on any empty dot on the drawing area. You can select multiple colors. Clicking on an already colored dot with a different color will change it.
3. Use the **erase** tool to delete individual dots, or use the **clean** tool to delete the whole drawing area.
4. Once your drawing is ready, copy the JSON in the "Coordinates JSON" textbox with the **copy** button.
